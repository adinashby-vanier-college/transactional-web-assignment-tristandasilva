BeatBash

  • Tech Stack: ReactJS, Springboot, MongoDB, Firebase Authentication
  • View Live // View API Repo // View Client Repo

Built using a modern tech stack, BeatBash is an web application designed for music festival lovers to read and leave reviews of their favourite music festivals worldwide. This platform offers a responsive interface for reading and posting festival experiences.

The back end is powered by Spring Boot, which is used to develop the Rest API that fetches data from the MongoDB collection. The React front-end calls the Rest API to render dynamic content and authenticate users.
